# kmu_rock_paper_scissors
국민대학교 네트워크최신기술 스마트 컨트랙트를 활용한 웹 서비스입니다.

Metamask 계정에 연결하여 0.2 이더리움을 지불해 컴퓨터와 가위 바위 보 게임을 할 수 있는 웹 서비스입니다.
